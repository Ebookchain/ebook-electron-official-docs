Introduction

Vérifiez que vous utilisez la bonne version de la documentation. Le numéro de version devrait faire partie de l'URL de la page. Si ce n'est pas le cas, vous utilisez probablement la documentation d'une branche de développement qui peut contenir des changements API qui ne sont pas compatibles avec votre version d'Electron. Si c'est le cas, vous pouvez changer de version sur la liste versions disponibles, ou, si vous utilisez l'interface de GitHub, ouvrez la liste déroulante "Switch branches/tags" afin de sélectionner le tag de votre version.
