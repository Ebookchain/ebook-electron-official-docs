SSS(Sıkça Sorulan Sorular)

Lütfen kullandığınız dokümanın Electron versiyonunuzla aynı olduğundan emin olun. Versiyon numarası okuduğunuz dokümanın URL'sindekiyle aynı olmalı. Eğer aynı değilse, muhtemelen geliştirme aşamasındaki API değişikliklerini içerebilecek dokümantasyonudur. Eğer öyleyse, atom.io üzerinden mevcut sürümlere göz atabilirsiniz ya da eğer GitHub arayüzünü kullanıyorsanız "Switch branches/tags" açılır menüsünden versiyonunuza uygun olanı seçebilirsiniz.
